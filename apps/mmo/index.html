    Canvas, SVG

    SVG

    • XML
    • Векторный
    • Retained mode

    Canvas

    • JS API
    • Растровый
    • Immidiate mode
    • Низкоуровневый доступ к пикселям

    SVG vs Canvas

    SVG

    • Нужно масштабирование
    • Надо привязывать события

    Canvas

    • Четкий контроль за пикселями
    • Много объектов
